We're excited to introduce **HarneSDK** — a Python SDK that lets you run AI agents like Claude Code programmatically inside secure, isolated sandboxes powered by [E2B](https://e2b.dev).

## What problem does it solve?

Running AI agents in production is tricky. You need to:

- **Isolate** agent execution so rogue code can't escape to your host system
- **Stream** output in real time without waiting for the full result
- **Serve** web applications generated by agents and expose them publicly
- **Automate** complex multi-step workflows without manual intervention

HarneSDK handles all of this with a clean, async-first Python API.

## How it works

Each agent session runs inside an E2B micro-VM. The SDK wraps the agent lifecycle — from spinning up the sandbox, to streaming stdout, to tearing everything down when you're done.

```python
from harnesdk.agent import AgentSession

async with AgentSession() as session:
    result = await session.run("Create a REST API in Python")
    print(result.output)
```

The `async with` pattern ensures sandboxes are always cleaned up, even if something goes wrong.
