Quickstart

Up and running in under two minutes.

1. Install

pip install harnesdk

2. Set API keys

export ANTHROPIC_API_KEY=...
export E2B_API_KEY=...
Run an agent and get output
import asyncio
from harnesdk.agent import AgentSession

async def main():
    async with AgentSession() as session:
        result = await session.run("Create a hello world HTTP server in Go")
        print(result.output)

asyncio.run(main())
Stream output in real time
async with AgentSession() as session:
    async for chunk in session.stream("Build a REST API in Python"):
        print(chunk, end="", flush=True)
Serve an app from the sandbox
async with AgentSession() as session:
    async for chunk in session.stream(
        "Build a dashboard and serve it on port 8000. "
        "Use: nohup python -m http.server 8000 > /tmp/s.log 2>&1 < /dev/null &"
    ):
        print(chunk)

    url = session.sandbox.get_host(8000)
    print(f"Live at https://{url}")

Security notice

Use short-lived, budget-limited API keys. AI agents running inside a sandbox may be susceptible to prompt injection attacks that attempt to exfiltrate credentials. We are actively working on hardened credential isolation.
